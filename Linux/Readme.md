### Commands I've learnt:
**Navigation**
1. print working directory
``` shell
pwd
```
2. list all files (and hidden) in a long format
``` shell
ls -l -a
ls -a ../
```
3. change directory
``` shell
cd
cd ..  #back one directory
cd ../.. # back 2 directories
cd ~ # cd back to user
cd / # cd back to root (parent of all parents :) )
```
